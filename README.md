# STEPS TO RUN

using dosbox 
mount your drive path of the files
get into the mounted drive
+ next use `masm /a pro.asm` 
+ skipe the object filename by typing `;` 
+ again skipe the run file by typing `;` 
+ enter the game by typing `pro`
 the controls are :
 `w` for up
 `s` for down
